# 签证进度追踪 📊

## 进度追踪工具 📈

### 添加签证申请

::: tip 申请信息
- 选择国家/地区
  - 申根签证
  - 美国签证
  - 日本签证
  - 其他...
- 填写申请日期
- 选择当前状态
:::

### 申请状态
- 🟡 材料准备中：收集所需文件
- 🟠 已预约面签：等待面签日期
- 🔵 已递交申请：等待审理
- 🟣 审理中：使馆处理中
- 🟢 已完成：签证办理完成

## 处理周期参考 ⏱️

::: info 常见签证处理时间
- 申根签证：约2-3周
- 美国签证：约2-4周
- 日本签证：约5-7个工作日
- 澳洲签证：约2-4周
:::

## 状态提醒 📱

### 自动提醒
::: tip 提醒设置
- 面签日期提醒

- 签证到期提醒

:::

### 进度更新
::: warning 注意事项
- 及时更新申请状态
- 记录重要时间节点

:::

## 使用指南 📖

1. 选择申请的签证类型
2. 填写申请基本信息
3. 上传相关材料扫描件
4. 设置提醒时间
5. 实时更新申请状态
